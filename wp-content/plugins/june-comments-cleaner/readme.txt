Tags: comments, spam, security, javascript
Requires at least: 2.7.1
Tested up to: 3.2
Stable tag: 1.3

This plugin will strip out all html tags(except the allowed ones), css attributes and javascript code from comments.


== Description ==
This plug-in prevents your blog visitors from adding malicious code to their comments by
stripping out all html, css, javascript tags from comments, except those you allow in the plugin's page, thus enhancing your blog security.


== Installation ==

1. Unzip the archive into the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to `Admin Panel > Settings > WTC Comment Cleaner` and add the tags you allow in your comments.
